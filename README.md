<h4>TARge25</h4>

Bitcoin calculator created using Avalonia - a workaround for creating WinForms on Linux.
This app uses CoinDesk API to access the required currency rates.

<h4>[CoinDesk Docs](https://developers.coindesk.com/documentation/data-api/introduction)</h4>
